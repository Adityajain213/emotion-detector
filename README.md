# Emotion Detector Web Application

This project is an AI-based web app that detects emotions from user input text using IBM Watson NLP.

## Functionalities
- Emotion detection (joy, anger, fear, disgust, sadness)
- Dominant emotion identification
- Flask web deployment
- Error handling for invalid input
- Unit testing
- Static code analysis

## Run Instructions
1. Install dependencies:
   pip install flask requests

2. Run application:
   python server.py

3. Open browser:
   http://127.0.0.1:5000
